# Classical Arabic Computational Linguistics

## An evolving computational linguistics framework for constructing, curating, and analyzing the Arabic hadith corpus of *Kanz al-Ummāl*

This repository documents an ongoing effort to construct a computational corpus of the approximately **46,600 hadith reports** preserved in *Kanz al-Ummāl* and to develop reproducible computational methods for their quantitative analysis.

The project combines corpus construction, computational linguistics, and statistical text analysis to explore the language of the hadith reports preserved in the largest Arabic hadith literature compilation.

## Current Status

The current Version 1.0 corpus demonstrates that large-scale computational analysis of *Kanz al-Ummāl* is feasible.

The preliminary pipeline includes:

- construction of a computational corpus from the Arabic text;
- descriptive corpus statistics;
- lexical frequency analysis;
- repeated phrase (n-gram) analysis;
- Arabic particle (ḥurūf) analysis;
- source abbreviation analysis; and
- preliminary narrator extraction.

The repository currently contains the rendered project documentation describing the corpus, computational framework, preliminary results, and future research directions.

## Why Collaboration Is Needed

The computational framework has reached the proof-of-concept stage. The principal challenge is now **corpus curation**.

Although much of the corpus has been parsed successfully, reliable computational analysis ultimately depends upon a carefully validated mapping between:

- hadith number,
- report text (*matn*), and
- source citations.

Improving this curated reference table is the highest priority for the project and will provide the foundation for subsequent computational linguistic analyses.

## Contributions

Contributions are welcome from anyone interested in:

- hadith studies;
- Classical Arabic;
- Arabic computational linguistics;
- corpus construction; or
- digital text curation.

The highest current priority is improving the underlying corpus by validating the relationship between hadith numbers, *matn*, and source citations.

## Future Directions

Once a fully curated corpus has been established, the project aims to support a broad range of computational investigations, including:

- corpus linguistics;
- formulaic language;
- Arabic morphology;
- stylometry;
- narrator and source networks;
- semantic analysis; and
- other quantitative approaches to Arabic hadith literature.
